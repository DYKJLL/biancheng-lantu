# 编程蓝图

让AI编程从"越写越乱"变成"有序构建"的核心方法论。

## 解决什么问题

用AI写代码经常遇到：代码越改越乱、改一处坏三处、报错不知道怎么修、效率越来越低。

这个技能让AI在动手之前先搞清架构、拆好模块、约定接口，按模块逐步推进，从一开始就避免混乱。

## 核心内容

- **三大场景**：新项目开发 / 改代码前判断 / 报错排查
- **四招架构思维**：模块化拆分 / 三层防御 / 先画蓝图 / 签对接合同
- **懒人通道**：简单项目（< 2小时）用精简流程

## 文件说明

```
编程蓝图/
├── SKILL.md      ← 技能主文件（给AI看的）
├── skill.json    ← 元数据（仓库信息）
└── README.md     ← 说明文档
```

## 安装方法

把整个文件夹放到 AI 的技能目录，重启 AI 即可。

## 快速开始

对AI说：
> "帮我写个登录功能"

AI会自动激活这个技能，先问你：
- "你想做什么系统？"
- "核心功能是什么？"

确认蓝图后再开始写代码。

## 来源

灵感来自使用AI编程时的真实痛点。
