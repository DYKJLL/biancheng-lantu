# 编程蓝图

让AI编程从"越写越乱"变成"有序构建"的核心方法论。

## 这是什么

一个AI技能文件。当你要用AI写代码时，它会让AI先画蓝图再动手，而不是上来就哐哐写。

基于九目AI的《AI帮你写代码却越写越乱？因为你少了这张蓝图》视频提炼而来。

## 核心内容

- **三大场景**：新项目开发 / 改代码前判断 / 报错排查
- **四招架构思维**：模块化拆分 / 三层防御 / 先画蓝图 / 签对接合同
- **懒人通道**：简单项目（< 2小时）用精简流程

## 文件说明

```
编程蓝图/
├── SKILL.md      ← 技能主文件（给AI看的）
├── skill.json    ← 元数据（仓库信息）
└── README.md     ← 说明文档
```

## 安装方法

把整个文件夹放到 AI 的技能目录，重启 AI 即可。

## 快速开始

对AI说：
> "帮我写个登录功能"

AI会自动激活这个技能，先问你：
- "你想做什么系统？"
- "核心功能是什么？"

确认蓝图后再开始写代码。
